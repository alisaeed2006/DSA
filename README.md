🚀 Data Structures & Algorithms Visualizer

An interactive web application for visualizing Data Structures and Algorithms step-by-step with smooth animations and clear explanations.

📌 Overview

This project is designed to help students and developers deeply understand how data structures and algorithms work internally through real-time visualization.

Instead of just reading code, users can see how operations happen step-by-step.

---

✨ Features

🔢 Sorting Algorithms

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort (with detailed splitting & merging visualization)
- Quick Sort (pivot = first element)
- Heap Sort
- Counting Sort

✔ Step-by-step animation
✔ Comparisons & swaps tracking
✔ Time & space complexity display

---

🧱 Data Structures

📦 Arrays

- Insert / Delete / Search
- Dynamic size control
- Clean cell-based visualization

🔗 Linked Lists

- Singly, Doubly, Circular, Ordered
- Insert (Head, Tail, Middle)
- Delete by value
- Fully animated pointer updates

📚 Stack & Queue

- Push / Pop / Peek
- Enqueue / Dequeue
- Correct visualization (no bars)

---

🌳 Trees

- Binary Tree
- Binary Search Tree (BST)
- AVL Tree (with rotations)
- Heap Tree

✔ Traversals (DFS, BFS)
✔ Insert / Delete / Search
✔ Step-by-step animations

---

🌐 Graphs

- BFS & DFS
- Dijkstra’s Shortest Path

✔ Dynamic graph generation
✔ Node & edge animations
✔ Clear path visualization
